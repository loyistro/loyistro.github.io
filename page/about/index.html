关于我

我是 Loyistro，主要是从中文「陸壹士多」衍生出来的谐音，本意是 一个叫陸壹拾的人开的一家陈列想法和经历的士多店 ，然后就歪成了现在这个名称，不过我倒挺喜欢的。

INFJ，一个对解谜、奇幻、工具略有兴趣的产品人。

我不确定自己对写作是否有兴趣和依赖，倒是有写了十几年的日记，虽然现在并没有很强的热情去频繁地写，但我总会想起一个场景：高中宿舍里住着的几个人各自躺在床上，恣意散发着对虫洞、思想、命运等不着边际的话题的暇想。

或许这代表着我还是很怀念当年那个不断涌现想法的自己，于是在文字、声音、视频这几种方式中，选了应该算是最方便记录和回顾的文字，在互联网环境倒退的现在，开始用回了博客。

博客建站

大概每个博客人第一篇文章都是自己建站的相关记录吧，我的过程也没有什么新意，就不单独写了简单说下就好：

  • 博客框架：Hugo + Github Pages + Github Action 部署的，凭几个关键字你就能找到很多资料了；
  • 主题：在 Hugo 主题官网看到的 no-style-please，在 ChatGPT 帮助下根据需要修改了一版放在了 Github 上，喜欢可以自取，感谢 AI 帮助零代码基础的我实现第一个项目；
  • 字体：使用国内免费的中⽂字体公益 CDN 服务 字图 CDN 提供的 京華老宋体，主要现在这个阶段我蛮喜欢复古；
